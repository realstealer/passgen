Как пользоваться:
===================================
запустили, кликнули - и пароль в буфере обмена

Программа позволяет:
  1) выбрать необходимую длину пароля; 
  2) набор символов, участвующих в генерации;
  3) алгоритм генерации - полный рандом или в удобной для запоминания форме;
